#  Mini Golf PRO: Ultimate Tournament

Benvingut a **Mini Golf PRO**, una experiència de simulació de golf completament desenvolupada en JavaScript i HTML5 Canvas. Supera nivells desafiants amb físiques realistes i entorns dinàmics.

##  Característiques Principals
* **Motor de Física Realista:** Moviment basat en vectors, fricció i col·lisions precises.
* **Tres Mons Únics:** *  **Valle de Esmeralda:** El bosc clàssic per començar.
  *  **Portes de l'Infern:** Obstacles de lava i entorn volcànic.
  *  **Dimensió Eterna:** Un buit espacial amb gravetat i estètica neó.
* **Sistema de Puntuació:** Gestió de cops (Strokes) i Par per cada nivell.
* **Visuals Avançats:** Partícules, il·luminació global i decoració animada procedimental.

##  Com Jugar
1. **Apunta:** Fes clic sobre la bola i arrossega el ratolí en la direcció oposada a la que vols disparar.
2. **Potència:** Com més lluny arrosseguis, més força tindrà el tir.
3. **Objectiu:** Posa la bola al forat amb el mínim de cops possibles.

## Tecnologia
* **Llenguatge:** JavaScript (Vanilla ES6)
* **Renderitzat:** HTML5 Canvas API
* **Estils:** CSS3 amb fonts de Google Fonts
